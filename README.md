## Deskripsi Website
Website ini merupakan aplikasi berbasis web yang dibangun menggunakan HTML, CSS, dan JavaScript, serta dijalankan menggunakan Node.js (Express). Website ini menggabungkan dua fitur utama:

---

##  1. Persona Finder
Fitur interaktif membantu pengguna untuk:

* Menginput nama lengkap
* Memilih hobi utama
* Memasukkan angka keberuntungan (1–10)

Sistem akan menghasilkan persona digital unik berdasarkan kombinasi input tersebut, lengkap dengan ikon visual dari Font Awesome.


##  2. Profile (Linktree Style)
Website juga menampilkan kartu profil pribadi dengan gaya Linktree, yang berisi:

* Foto profil
* Nama pengguna (dinamis sesuai input)
* Deskripsi singkat
* Tombol link full-width ke berbagai platform:

  * GitHub
  * LinkedIn
  * Instagram
  * Email


##  Fitur Tambahan

* Data tersimpan menggunakan localStorage
* Persona tetap tampil meskipun halaman di-refresh atau kembali dari link
* Tombol Reset untuk menghapus data dan mengulang input
* Animasi dan desain modern (glassmorphism + baby blue theme)
